Extension Name: DigDog

Short Description :
Instant information about any website you visit - location, ownership, server details, and more with one click.

Detailed Description:

Ever visited a website and wondered who's behind it, where it's hosted, or what technology it runs on? This extension gives you instant answers with just one click.

Simply click the extension icon while browsing any website, and you'll get comprehensive information including:

#Server location (country, city, and coordinates)
#Domain ownership details (registrar, registration date, expiry date)
#Server technology (web server software, x-powered-by headers)
#Network information (ISP, IP address)

Whether you're a security researcher investigating suspicious sites, a developer curious about how your favorite websites are built, a marketer researching competitors, or just someone who likes to understand more about the internet, this tool saves you from switching between multiple websites and tools.

Everything is presented in a clean, easy-to-read format right in your browser.

Features:

One-click lookup for any website

Clean and intuitive interface

No sign-up or login required

Privacy-focused - no data collection

Open source on GitHub
